⚔️ Serveur World of Warcraft sous Docker

BTS SIO 1ère année – TP Administration Système | Janvier 2026

📋 Description du projet

Déploiement d'une infrastructure conteneurisée complexe via Docker Compose sur une VM Debian. Le serveur AzerothCore (WoW 3.3.5a) repose sur trois conteneurs orchestrés : le WorldServer, l'AuthServer et une base de données MySQL. Administration complète via ligne de commande.

⚙️ Compétences mises en œuvre

Docker Docker Compose Linux Debian MySQL AzerothCore Administration réseau mysqldump VirtualBox

✅ Réalisations principales

Installation de Docker et Docker Compose · Déploiement des conteneurs WorldServer, AuthServer et Database · Création de comptes joueurs et attribution des privilèges Game Master (niveau 3) · Vérification en base MySQL · Configuration du fichier realmlist.wtf · Sauvegarde complète via mysqldump · Extraction des fichiers de configuration critiques.
